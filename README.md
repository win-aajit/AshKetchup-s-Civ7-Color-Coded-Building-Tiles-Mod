# AshKetchup-s-Civ7-Color-Coded-Building-Tiles-Mod

This mod should make it easier to keep track of your adjacencies and where your buildings of each type are while still being subtle, making the sprawl more readable.

There are options in the addons menu to customize the opacity level of fill color, and options to choose render mode between fill color only, edge color only, or both. Can also be toggled on/off within the Lenses/Decorations menu.

Color is currently based on the highest yield type of a tile with a building on it.

Works with ToT 1.4.0
